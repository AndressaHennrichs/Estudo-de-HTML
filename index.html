HTML(Linguagem de Marcação de HiperTexto) é o bloco de construção mais complexo básico da web, ela nao sen enquadra como linguagem de programação1, mas como uma linguagem de marcação. Hipertexto são os links que conectam paginas e Marcação sendo usado para anotar, textos, imagens e etc, para exibir em um navegador. Ou seja, HTML define o significado e a estrutura do conteúdo da web.

Elementos HTML

(Voltar)

Elementos HTML2 são quaisquer objetos identificáveis dentro de um documento, como um caractere, palavra, imagem, parágrafo ou célula de planilha. Levando-se em conta os graus de importância dos assuntos, temos como exemplos de titulos:

    Sendo a Tag Título de maior importância, é utilizada apenas uma vez por site

Atributos

(Voltar)

Atributos são informações que passamos na Tag para que ela se comporte da maneira esperada. Existem atributos globais e específicos. A diferença de uma Tag e um Atributo seria:

"uma tag altera a maneira que o elemento aparece e um atributo descreve as características desse elemento"
